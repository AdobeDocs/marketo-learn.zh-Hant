# 組織新專案的最佳實務 [!DNL Marketo Engage] 例項

當您第一次開始使用Marketo Engage時，組織是關鍵。 在執行個體中保持井然有序，讓您和您的同事可以輕鬆追蹤行銷活動、方案和資產。 瞭解如何在中組織導覽樹狀結構 [!UICONTROL 行銷活動] 使用資料夾、標準命名慣例和功能（例如複製），輕鬆與您的同事合作使用新的 [!DNL Marketo Engage] 執行個體。
