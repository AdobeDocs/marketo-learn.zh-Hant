# 對話式登陸頁面

只要按幾下，即可將Dynamic Chat交談流程安裝至任何登陸頁面(Marketo登陸頁面或自訂)，並轉換任何連結或按鈕以觸發交談參與。

您可以解除鎖定的使用案例 — 

- 啟用訪客預約會議
- 提出資格問題
- 根據訪客選擇/興趣的建議內容
- 共用檔案及追蹤參與
