# 如何稽核繼承的Marketo Engage執行個體

無論您是經驗豐富的管理員還是新晉管理員，要知道從哪裡開始清理都很困難，繼承Marketo Engage執行個體可能會相當困難。 首先，建議您花時間瞭解您今天繼承的執行個體內容，並建立更新設定的行動計畫。

在本教學課程中，我們將逐步引導您瞭解Marketo Engage中最重要的區段，包括管理、行銷活動、設計工作室和資料庫，並在此過程中為您提供稽核和最佳化的秘訣。
