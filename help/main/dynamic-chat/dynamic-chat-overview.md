# Dynamic Chat 教學課程

* 概觀
   * **[快速產品導覽[2023]](product-tour.md)**

     *逐步指南可協助您在短短 15 分鐘內透過 [!UICONTROL Dynamic Chat]  上線並進行首次對話！*
* 設定
   * **[設定並安裝您的聊天機器人](setup.md)**
     *了解設定 [!UICONTROL Dynamic Chat] 訂閱的步驟。*
   * **[啟用您的Marketo整合，透過[!UICONTROL Dynamic Chat]](marketo-integration.md)**
     *透過交談鎖定您的網頁訪客，並利用 Marketo 的銷售線索資訊打造個人化的交談內容。*
   * **[新增和管理使用者[!UICONTROL Dynamic Chat]](user-management.md)**
     *了解如何在 [!UICONTROL Dynamic Chat] 中新增或管理使用者。*
   * **[代理程式管理](agent-management.md)**
     *了解如何管理代理。*

   * **[建立和管理個人化對話](dialogue-management.md)**

   * **[對話方塊預覽](dialogue-preview.md)**
     *利用對話預覽功能，可在發佈前輕鬆預覽您的對話。*

   * **[將訪客連線到即時代理程式](connect-visitors-to-live-agents.md)**
